# Сравнительный анализ алгоритмов поиска пути в графе: алгоритм A* (A-star) и алгоритм поиска в глубину
В этой работе будет произведен сравнительный анализ двух алгоритмов поиска пути в графе: алгоритм A* (A-star) и алгоритм поиска в глубину.
<h1>Особенности</h1>
<p>Реализация двух алгоритмов сортировки на C++</p>
<p>Реализация функции для экспериментального тестирования</p>
<p>Использование высокоточного таймера (chrono)</p>
